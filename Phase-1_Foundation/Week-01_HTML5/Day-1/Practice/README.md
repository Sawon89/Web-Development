# 📝 Day 1 — Practice Area
## তোমার কাজের জায়গা

---

## এই folder এ কী করবে?

এই `Practice` folder টা শুধু **তোমার জন্য** — এখানে তুমি lab এর tasks করবে।

---

## কীভাবে কাজ করবে?

1. **`my-work.html`** file টা VS Code এ খোলো
2. **`../LAB.md`** দেখে একটা একটা করে task করো
3. কাজ শেষে **আমাকে বলো** — "Day 1 lab শেষ করেছি"
4. আমি তোমার `my-work.html` **check করব**
5. সব ঠিক থাকলে → Day 2 পাবে ✅

---

## ⚠️ Rules

- Placeholder গুলো (`[তোমার নাম লেখো]`) **নিজের তথ্য দিয়ে** replace করো
- LAB.md এর **সব task** করতে হবে
- কোনো সমস্যা হলে আমাকে **জিজ্ঞেস করো** — এগিয়ে যাওয়ার আগে বোঝো
