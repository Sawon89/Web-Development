## Project Requirements

তোমার পোর্টেফোলিওতে নিচের জিনিসগুলো **অবশ্যই থাকতে হবে**:

1. **Header** — তোমার নাম, title, এবং navigation লিংক।
2. **About Section** — তোমার ছবি (Figure ও Figcaption সহ), তোমার পরিচয়, এবং personal details।
3. **Skills Section** — Table দিয়ে তোমার স্কিলগুলো দেখাও।
4. **Projects Section** — তোমার ৩টা প্রজেক্টের নাম (যেগুলো তুমি Day 1, 2, 3 এ করেছো) লিস্ট আকারে দেখাও।
5. **Contact Section** — একটা সম্পূর্ণ ফর্ম (নাম, ইমেইল, মেসেজ, সাবমিট বাটন) এবং সোশ্যাল মিডিয়া লিংক (target="_blank" সহ)।
6. **Footer** — Copyright লেখা।
7. **Semantic Tags** — `header`, `nav`, `main`, `section`, `article`, `aside`, `footer` ইত্যাদি সঠিকভাবে ব্যবহার করতে হবে।
8. **Accessibility** — `alt` text, `aria-label`, ফর্মের `label` এবং কীবোর্ড নেভিগেশন (Skip to content) থাকতে হবে।

## Step-by-Step Guide

### Step 1: File Setup করো
এই `Mini-Project_Portfolio` ফোল্ডারে `index.html` ফাইলটা খোলো।
Basic HTML structure লেখো।

### Step 2: Semantic Structure তৈরি করো
Day 4-এর মতো `header`, `main`, `footer` দিয়ে কাঠামো তৈরি করো।

### Step 3: Content বসাও
প্রতিটি সেকশনের ভেতর প্রয়োজনীয় কনটেন্ট, ফর্ম এবং টেবিল বসাও।

### Step 4: Accessibility চেক করো
সব ছবি এবং লিংকে `alt` এবং `aria-label` আছে কিনা চেক করো।
