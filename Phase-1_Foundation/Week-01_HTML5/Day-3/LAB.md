## Step 2: Tasks

### Task 1 — Basic Structure ও Title
Basic HTML structure লেখো এবং `<title>`-এ "Contact Form" দাও। `<body>`-এর ভেতরে কাজ শুরু করো।

### Task 2 — Learning Schedule Table
একটা রুটিন তৈরি করো (যেমনটা `STUDY.md`-তে দেখেছো)। 
- `<table>` এর ভেতর `<thead>` এবং `<tbody>` ব্যবহার করো।
- ৩-৪ টা কলাম রাখো: দিন (Day), টপিক (Topic), সময় (Time)।
- টেবিলটি দেখতে সুন্দর হওয়ার জন্য `<table border="1">` ব্যবহার করতে পারো।

### Task 3 — Contact Form তৈরি
টেবিলের নিচে একটা Contact Form বানাও (`<form>`)। এর মধ্যে নিচের ইনপুটগুলো রাখো:
- **Full Name:** একটা Text input দাও। (label এবং `required` ব্যবহার করবে)
- **Email:** একটা Email input দাও। (`placeholder="email@example.com"` ব্যবহার করবে)
- **Age:** একটা Number input দাও।
- **City:** একটা Select dropdown দাও (ঢাকা, খুলনা, ইত্যাদি)।
- **Interests:** অন্তত ২-৩টা Checkbox দাও (যেমন: HTML, CSS, JavaScript)।
- **Experience Level:** Radio buttons দাও (Beginner, Intermediate, Advanced)।
- **Message:** একটা Textarea দাও (User এর মেসেজ লেখার জন্য)।
- **Submit Button:** একটা Submit বাটন দাও।

## ✅ Checklist (নিজে Check করো)

- [ ] HTML এর basic structure আছে
- [ ] একটা `<table>` আছে, যাতে `thead`, `tbody`, `tr`, `th`, `td` আছে

- [ ] Form-এ `checkbox` ও `radio` বাটন আছে

- [ ] প্রতিটি input-এর সাথে সঠিক `label` আছে (`for` ও `id` মিলিয়ে)
