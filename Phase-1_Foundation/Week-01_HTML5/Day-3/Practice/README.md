# 📝 Day 3 — Practice Area

---

## এই folder এ কী করবে?

এই `Practice` folder টা শুধু **তোমার জন্য** — এখানে তুমি Day 3 এর lab এর tasks করবে।

---

## কীভাবে কাজ করবে?

1. **`my-work.html`** file টা VS Code এ খোলো (এটা এখন প্রায় খালি আছে)
2. **`../LAB.md`** দেখে একটা একটা করে task করো
3. কাজ শেষে **আমাকে বলো** — "Day 3 lab শেষ করেছি"
4. আমি তোমার `my-work.html` **check করব**
5. সব ঠিক থাকলে → Day 4 পাবে ✅
