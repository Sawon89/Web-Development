## Lab এর লক্ষ্য

আজ তুমি একটা **Blog Post Page** তৈরি করবে যেখানে থাকবে:
- Navigation links
- একটা Article (Text)
- Related links এর list
- একটা Image

## Step 1: Practice File খোলো

📂 এই path এ যাও: `Day-2/Practice/my-work.html`
এই file টা খালি আছে, আগের দিনের মতই `<!DOCTYPE html>` থেকে লেখা শুরু করো।

---

## Step 2: Tasks (একটা একটা করো)

### Task 1 — Basic Structure
HTML এর basic structure (`html`, `head`, `title`, `body`) লেখো। Title এ তোমার পছন্দ মত একটা নাম দাও।

### Task 2 — Navigation Menu (Links)
`<body>` এর একদম শুরুতে কিছু links দাও:
- Home (href="#")
- About (href="#")
- Contact (href="#")
- (Bonus: একটা link এ `target="_blank"` দিয়ে google.com এ যাওয়ার link দাও)

### Task 3 — Blog Post Headline
`<h1>` দিয়ে তোমার Blog এর একটা Title দাও (যেমন: "কেন আমি Web Developer হতে চাই?")।
নিচে একটা `p` tag এ আজকের তারিখ লেখো।

### Task 4 — একটা Image যোগ করো
Online থেকে যেকোনো একটা image যোগ করো।
*Hint:* `<img src="https://picsum.photos/600/300" alt="Blog Image" width="600">` ব্যবহার করতে পারো।

### Task 5 — Blog Content
দুটো বা তিনটে `<p>` tag দিয়ে তোমার Blog এর main content লেখো।

### Task 6 — Learning List (Ordered List)
তুমি কী কী শিখবে তার একটা Numbered List (`<ol>`) তৈরি করো।
যেমন: 1. HTML, 2. CSS, 3. JavaScript

### Task 7 — Useful Links (Unordered List)
কিছু কাজের resources এর Bullet List (`<ul>`) তৈরি করো।
যেমন: MDN Web Docs, W3Schools।
*Bonus:* এই list এর item গুলোর ভেতরে `<a>` tag দিয়ে link করে দাও!
